# Number Row Characters
- `!` Exclamation Mark
- `@` "At" Symbol
- `#`  Hashtag
- `$` Dollar Sign
- `%` Percent Symbol
- `^` Up Arrow (Sometimes used to create exponents)
- `&` And Symbol
- `*` Asterisk
- `(` Open Bracket
- `)` Closed Bracket
